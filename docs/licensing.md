# Licensing Notes

This document explains how licensing is handled in this repository.

## Core source code

The core source code is licensed under Apache License 2.0.

SPDX identifier:

```text
Apache-2.0
```

## Third-party dependencies

Third-party dependencies retain their own licenses.

See:

```text
THIRD_PARTY_LICENSES.md
```

## Language dictionaries and models

Language dictionaries, embeddings and model weights may have separate licenses and are not vendored into this repository unless explicitly stated.

Do not commit downloaded dictionaries, model archives, embeddings, weights or generated model caches without a separate license review.

## Synthetic data

The public synthetic regression corpus is distributed with the repository under Apache-2.0 at this stage.

Synthetic examples must not contain real personal data.
