# Disclaimer

BeforeSending is an experimental local-first pseudonymization / reversible masking / risk-reduction tool for preparing documents before external AI/SaaS use.

It does **not** guarantee complete anonymization, legal compliance, or the absence of all personal or confidential data in the output.

This project is not:

- a certified anonymization tool;
- a 152-FZ/GDPR/HIPAA compliance solution;
- an enterprise DLP system;
- a substitute for human review;
- a tool for scanned PDFs, images or OCR in v0.1.

The tool may produce false positives and false negatives. Always review the generated output and reports before sharing documents with third parties or external AI/SaaS services.

If a token dictionary such as `project_dictionary.json` is preserved, the process is reversible pseudonymization, not irreversible anonymization.

Do not share, upload or commit real input documents, token dictionaries, review reports, feedback files or manual rule files containing real personal or confidential data.
